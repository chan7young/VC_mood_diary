# 🌙 무드 다이어리 (Mood Diary)

> 일기를 쓰면 LLM이 감정과 반복 주제를 읽어내고, 시간에 따른 변화로 시각화해 주는 **자기 이해 보조 도구**.

⚠️ 이 앱은 성찰을 돕는 도구이며, **의학적 진단이나 전문 상담을 대체하지 않습니다.**

---

## 🎯 프로젝트 소개

- **독창성:** 일기를 "쓰기"에서 끝내지 않고, 감정·반복 주제를 분석해 변화로 시각화하고, 위기 신호 시 상담 자원을 안내합니다.
- **목표:** 사용자가 자신의 감정·생각의 패턴을 스스로 관찰·이해하도록 돕습니다 (진단이 아닌 성찰).
- **기대효과:**
  1. 감정의 언어화·기록 습관을 통한 정서적 자기 인식 향상
  2. 감정 추이·반복 주제 시각화로 영향 요인 자각
  3. 위기 안내 설계로 한계를 인정하는 책임 있는 AI 활용 사례 제시

## ⚙️ 설치 및 실행

```bash
# 1) 가상환경(선택) 후 의존성 설치
pip install -r requirements.txt

# 2) API 키 설정 (없어도 mock 모드로 실행됩니다)
cp .env.example .env   # 그리고 .env 안에 본인 키 입력
#   키 발급: https://aistudio.google.com (무료, 신용카드 불필요)

# 3) 실행
streamlit run app.py
```

> 🔐 **보안:** 레포가 Public이므로 `.env`(실제 키)는 **절대 커밋하지 마세요.** 이미 `.gitignore`에 등록되어 있습니다. 팀원은 각자 자신의 무료 키를 로컬에서 발급해 사용하세요.

## 🤝 협업 규칙 (GitHub)

1. 각자 자기 **브랜치**에서 작업합니다. (예: `feat/llm-analysis`)
2. 작업이 끝나면 **PR(Pull Request)** 을 올립니다.
3. 다른 팀원이 **리뷰·승인**한 뒤 `main`에 머지합니다. (main 직접 push 금지)
4. AI에게 쓴 프롬프트는 `prompts/PROMPTS.md`에 누적 기록합니다.
