# assets

README 등 문서에서 사용하는 이미지(스크린샷, GIF)를 보관하는 폴더입니다.

## 실행 화면 추가 방법

1. 앱을 실행합니다.
   ```bash
   streamlit run app.py
   ```
2. 화면을 캡처해 아래 이름으로 이 폴더에 저장합니다.
   - `diary.png` — 일기쓰기 화면
   - `dashboard.png` — 감정 대시보드 화면
   - (GIF는 `demo.gif` 등 자유롭게)
3. 커밋·푸시하면 루트 `README.md`의 "실행 화면" 표에 자동으로 나타납니다.
